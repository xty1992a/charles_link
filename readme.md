## 简介
使用Charles抓包时，需要将主机ip/port手动填写到手机端。  
借助`Shadowrocket`等工具，使用http代理可以避免这个麻烦。
本项目是一个utool插件，用于快速生成一条shadowrocket代理链接，扫码即可用。

## 使用
0. utool安装本插件
1. 主机开启Charles或其他抓包工具。
2. 呼出utool，使用`抓包`关键字唤起插件。
3. 手机打开`Shadowrocket`,扫码导入代理。
